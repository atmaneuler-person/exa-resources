# Introduction

Welcome to the <b>EXA Solution Documentation</b>.

This section works differently from the blog.

- The <b>Left Sidebar</b> shows all available documentation pages.
- Each file you create in the `Docs` folder becomes a menu item.

## How to add a new page?

Simply add a new `.mdx` file in `data/posts/[locale]/Docs/`.

Example: `data/posts/ko/Docs/getting-started.mdx`
