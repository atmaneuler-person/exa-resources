# Introduction

Welcome to the **EXA Solution Documentation**.

This section works differently from the blog.

- The **Left Sidebar** shows all available documentation pages.
- Each file you create in the `Documentation` folder becomes a menu item.

## How to add a new page?

Simply add a new `.mdx` file in `data/posts/[locale]/Documentation/`.

Example: `data/posts/ko/Documentation/getting-started.mdx`
