# EXAWin이란?

**EXAWin**은 베이지안 통계 기반의 영업 확률 분석 시스템입니다.

영업 활동 데이터를 기반으로 **거래 성사 확률을 과학적으로 분석**하여, 영업팀이 어떤 거래에 집중해야 하는지 데이터 기반 의사결정을 지원합니다.

## 핵심 기능

- **베이지안 확률 분석** — 영업 시그널과 스테이지 데이터를 기반으로 거래 성사 확률을 실시간 계산
- **스테이지 관리** — 영업 프로세스의 각 단계를 정의하고 추적
- **시그널 관리** — 고객 반응 시그널을 체계적으로 분류하고 영향도 측정
- **액티비티 관리** — 미팅, 통화 등 영업 활동을 기록하고 분석
- **프로젝트(딜) 관리** — 거래별 진행 상황과 확률 변화를 한눈에 파악

## 시작하기

EXAWin을 시작하려면 아래 순서로 설정하세요:

1. **마스터 데이터 설정** — 채널, 스테이지, 시그널 마스터
2. **고객 및 컨택터 등록** — 고객사와 담당자 정보 입력
3. **프로젝트 등록** — 신규 영업 프로젝트(딜) 생성
4. **활동 기록** — 영업 미팅, 활동 내역 입력
5. **분석 확인** — 베이지안 분석 결과와 확률 추이 확인
