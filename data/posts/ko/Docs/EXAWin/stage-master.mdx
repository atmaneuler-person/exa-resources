# 스테이지 마스터

**스테이지 마스터**는 영업 프로세스의 각 단계를 정의하고 관리하는 설정 화면입니다.

## 스테이지란?

영업 프로세스는 일반적으로 여러 단계를 거칩니다:

1. **리드 발굴** — 잠재 고객 식별
2. **니즈 파악** — 고객 요구사항 분석
3. **제안** — 솔루션 제안서 전달
4. **협상** — 가격 및 조건 협상
5. **계약** — 최종 계약 체결

각 단계가 하나의 **스테이지**이며, 스테이지 마스터에서 이를 설정합니다.

## 화면 구성

스테이지 마스터 화면은 다음으로 구성됩니다:

- **스테이지 목록** — 등록된 모든 스테이지를 탭으로 표시
- **상세 정보** — 선택된 스테이지의 파라미터 설정
- **베이지안 파라미터** — SV(Stage Value), T(Threshold), k(Steepness) 값 설정

## 스테이지 추가

1. 스테이지 마스터 화면에서 **"추가"** 버튼을 클릭합니다
2. 스테이지 이름을 입력합니다
3. 정렬 순서(Sort Order)를 설정합니다
4. 베이지안 파라미터를 설정합니다:
   - **SV (Stage Value)** — 이 단계의 기본 가치 (0~100)
   - **T (Threshold)** — 시그모이드 함수의 임계값
   - **k (Steepness)** — 시그모이드 함수의 기울기
5. **저장** 버튼을 클릭합니다

## 주의사항

> ⚠️ 이미 프로젝트에서 사용 중인 스테이지를 비활성화하면, 해당 프로젝트의 액티비티 보드에서 해당 스테이지가 숨겨집니다.

- 스테이지 삭제는 불가능합니다. **비활성화**만 가능합니다.
- 비활성화된 스테이지는 신규 프로젝트에 복사되지 않습니다.
