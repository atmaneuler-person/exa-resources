## Bayesian MVP 시스템 개요

본 시스템은 비즈니스 현장에서 발생하는 다양한 데이터를 베이지안 방법론을 통해 분석하고, 최적의 의사결정을 내릴 수 있도록 돕는 최소 기능 제품(MVP)입니다.

### 주요 기능

1. **사전 지식(Prior) 설정**: 과거 데이터나 전문가의 견해를 기반으로 초기 확률을 설정합니다.
2. **데이터 업데이트 (Likelihood)**: 실제 현장에서 수집된 데이터를 입력하여 믿음을 갱신합니다.
3. **결과 시각화 (Posterior)**: 최종적으로 도출된 사후 확률을 그래프와 리포트로 확인합니다.

### 사용 방법

- 홈페이지 상단의 'MVP 연동' 버튼(예정)을 클릭하여 시스템에 접속합니다.
- 데이터 입력 폼에 분석하고자 하는 수치를 입력합니다.
- '분석 실행' 버튼을 누르면 베이지안 추론 과정이 진행됩니다.

자세한 수식 및 파이썬 코드 구현은 블로그의 관련 기술 아티클을 참조해 주세요.
